## 创建用户和目标文件夹

```bash
adduser appwrite
```

```bash
sudo mkdir /opt/appwrite/ && cd /opt/appwrite
```

## docker-cli 部署

[Appwrite 0.14 - Stack · GitHub](https://gist.github.com/eldadfux/977869ff6bdd7312adfd4e629ee15cc5)
首先打开两个端口，一个 http 端口（默认 80），一个 https 端口（默认 443），我这里选择 88 和 440（如果喜欢用默认设置可以不改）

```bash
sudo ufw allow 88
sudo ufw allow 440
```

运行 docker cli 安装：

```bash
docker run -it --rm \
    --volume /var/run/docker.sock:/var/run/docker.sock \
    --volume "$(pwd)"/appwrite:/usr/src/code/appwrite:rw \
    --entrypoint="install" \
    appwrite/appwrite:0.14.2
```

出现`Starting Appwrite installation` 时，依此输入 端口 80、端口 440，如果选默认的可以直接回车跳过，下面的 A 记录也可以留空，API Key 可以在这里随便生成一段，然后保存好：[在线加密解密](https://tool.oschina.net/encrypt)
![](https://usc1.contabostorage.com/cc0b816231a841b1b0232d5ef0c6deb1:image/2022/06/b59d2a33bd6fe72d62a0c9599813b754.png)

显示安装成功后配置 Nginx 反代
首先要在域名管理的地方添加一个域名的 A 记录，然后修改 Nginx 配置：

```bash
sudo  nano /etc/nginx/conf.d/appwrite.conf
```

```conf
server {
    listen 80;
    server_name example.com;

    location / {
        proxy_pass http://127.0.0.1:85;
        proxy_redirect off;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```
